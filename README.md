🤖 ChatMind AI Assistant



An advanced AI-powered chatbot and multi-agent assistant developed using Python, Flask, NLP, and Local LLM integration. This project combines Natural Language Processing, Machine Learning, and AI-based response generation to create an intelligent conversational assistant capable of handling user queries, generating content, and supporting future AI agent integrations.



🚀 Features



✅ NLP-based intent classification

✅ Machine Learning response prediction

✅ Flask web application interface

✅ Local LLM integration using Ollama

✅ Context-aware chatbot responses

✅ AI-powered content generation

✅ PDF and DOCX generation support

✅ Image generation request handling

✅ Download generated files

✅ Multi-agent architecture support

✅ Responsive chatbot UI

✅ Offline AI assistant capability

⚙️ Installation

📥 Clone Repository

git clone https://github.com/your-username/ChatMind.git

cd ChatMind

🧪 Create Virtual Environment

python -m venv venv

▶️ Activate Environment

🪟 Windows

venv\\Scripts\\activate

🐧 Linux / 🍎 Mac

source venv/bin/activate

📦 Install Dependencies

pip install -r requirements.txt

🧠 Install Ollama



Download and install Ollama:



Ollama Official Website



⬇️ Download Local AI Model



Run one of the following:



⚡ Phi3

ollama run phi3

🚀 TinyLlama

ollama run tinyllama

🧠 Mistral

ollama run mistral

▶️ Run the Application

python app.py



Application will start at:



http://127.0.0.1:5000
